Features

  • Fixes issue #1 of the version 1.0.0
  • Increase security with a .htaccess file created in each servers
  • Block any direct access to a server via a browser by asking for credentials
  • Use the class PreparedQueryStatement to manage prepared queries
  • Use the class Util to do common stuff (like encrypt strings...)
